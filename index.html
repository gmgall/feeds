Uma reclamação

Tudo seria mais simples se os sites simplesmente fornecessem feeds completos, mas sites que fazem isso tornam-se cada vez mais raros. A regra do jogo da web moderna é retenção de tráfego. Os sites precisam que nós não saiamos do feudo internético deles para terem mais oportunidades de nos mostrar anúncios.

Eu resisto a isso e consumo a web principalmente por feeds RSS. Os feeds que eu gero para sites que originalmente não têm feeds estão listados abaixo.
